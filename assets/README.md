# Assets Folder

This folder is for storing game assets such as:
- Images (icons, backgrounds, etc.)
- Sound effects (optional)
- Fonts (if not using web fonts)
- Other media files

Current assets:
- `idle_cat.gif` – Copy Cat waiting animation
- `typing_cat.gif` – energetic typing animation used during input

If you want to add sound effects for typing feedback or background music, place them here as well.
